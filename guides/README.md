# Site Makita - Affiliation AliExpress

Site multilingue d'affiliation pour les produits Makita.

## Langues disponibles
- Français (fr/)
- Allemand (de/)
- Espagnol (es/)
- Portugais (pt/)

## Structure
- `index.html` : Page d'accueil (racine)
- `fr/`, `de/`, `es/`, `pt/` : Sites par langue
- `images/` : Images partagées
- `scripts/` : Scripts de génération
- `sitemap.xml` : Sitemap index

## Déploiement
Déployé automatiquement sur Cloudflare Pages via GitHub.
